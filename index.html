A work of interactive fiction.

Turkey Street was created with Inform and has IFID CB0889D3-9A38-466A-AFFC-CEB1D35A3F4A. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
